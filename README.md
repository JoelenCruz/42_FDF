# 42_FDF
Representing a landscape as a 3D object in which all surfaces are outlined in lines.

## 1- Clone
	
	git clone https://github.com/JoelenCruz/42_FDF.git FDF_jcruz-da

    cd FDF_jcruz-da
   	make

    ./fdf test_maps/42.fdf
